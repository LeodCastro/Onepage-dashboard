# Onepage dashboard
Dashboard para acompanhamento do contrato

## Objetivo
Acompanhar as ocorrências de suporte e melhorias registradas através das horas das atividades lançadas.

## Ferramentas
Power BI
Dynamics 365 Customer Service
Dataverse
Onedrive
Planilhas Excel
